## Contenido temático del curso

### 1. Node.js y entorno de ejecución
#### 1.1 Node.js, npm y ejecución de JavaScript
Introducción a Node.js como entorno de ejecución fuera del navegador, uso básico de la terminal, inicialización de proyectos y primer script.

#### 1.2 Fundamentos de JavaScript orientados al backend
Variables, estructuras de datos, funciones, desestructuración y métodos de arrays aplicados al desarrollo backend.

#### 1.3 Asincronía en Node.js
Promesas, async/await, flujo asíncrono y modelo non-blocking I/O.

#### 1.4 Módulos nativos de Node.js
Uso de fs, path y os para manejo de archivos, rutas e información del sistema.

---

### 2. Gestión de dependencias y herramientas de ejecución
#### 2.1 ¿Qué es npm?
Gestión de dependencias y devDependencies, semver, resolución de versiones, node_modules y buenas prácticas.

#### 2.2 Uso de npx
Diferencias entre npm y npx, ejecución de paquetes sin instalación global y ventajas en entornos de desarrollo.

---

### 3. TypeScript para desarrollo backend
#### 3.1 Introducción a TypeScript
Qué es TypeScript, diferencias con JavaScript y ventajas de su uso en el backend.

#### 3.2 Configuración inicial de TypeScript en Node.jsc
Instalación de TypeScript, creación y configuración del archivo tsconfig.json, estructura del proyecto y compilación.

#### 3.3 Tipado en TypeScript
Tipos primitivos y básicos, interfaces, tipado de objetos, tipos de funciones y arrays tipados.

---

### 4. Servidores y frameworks backend
#### 4.1 Servidor HTTP básico con Node.js
Creación de un servidor HTTP manual y manejo de peticiones GET.

#### 4.2 Introducción a Express.js
Uso de Express como framework, creación de rutas básicas y comparación con servidor nativo.

#### 4.3 Validación de datos con Zod
Creación de esquemas y validación de datos de entrada.

### 4.4 Middlewares en Express
Concepto de middleware, uso de middlewares nativos y personalizados.
---

### 5. Contenedores y bases de datos
#### 5.1 Introducción a Docker
Conceptos básicos de contenedores e imágenes y administración esencial.

#### 5.2 Docker Compose
Configuración mínima de servicios, puertos y volúmenes.

#### 5.3 MySQL con Docker Compose
Montaje de MySQL con persistencia de datos.

#### 5.4 Preparación de la base de datos
Importación de scripts SQL y verificación de estructuras.

#### 5.5 Conexión a MySQL desde Node.js
Configuración de MySQL2 y pruebas de conexión.

---

### 6. Desarrollo de una API REST profesional
#### 6.1 Introducción a la arquitectura REST
Endpoints, métodos HTTP y códigos de estado.

#### 6.2 Estructura de un proyecto Express profesional
Organización en rutas, controladores y servicios.

#### 6.3 Creación de endpoints base
Implementación de rutas GET y POST con respuestas JSON.

#### 6.4 Validación de datos con Zod en la API
Validación de requests y manejo de errores.

#### 6.5 Integración con MySQL (capa Service)
Consultas, inserciones y manejo de errores.

#### 6.6 Pruebas de endpoints con Postman
Pruebas de flujos completos y casos de error.

---

### Resultado final del curso
Construcción de un CRUD completo integrando Node.js, TypeScript, Express, Zod, MySQL y Docker, siguiendo buenas prácticas profesionales.
